# Sobre:
Fiz esse projeto a partir dos conhecimentos em requisições AJAX. Nele, você pode adicionar qualquer CEP válido e irá retornar algumas as informações como Logradouro, Cidade e Estado.

-	CSS
-	Javascript
	-	Axios
